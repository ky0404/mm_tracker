## 三、5阶段判定模型

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          庄家操盘5阶段模型                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. 静默积累期                                                          │
│     特征: 价格横盘，成交量萎靡                                          │
│     动作: 监控，不入场                                                  │
│                                                                         │
│  2. 洗盘测试期                                                          │
│     特征: 价格突然拉升30%-80%后回落                                     │
│     动作: 等待，不入场                                                  │
│                                                                         │
│  3. 拉升启动期 ← 唯一入场阶段                                           │
│     特征: 突破关键位 + 放量 + 资金费率转正                               │
│     动作: 入场                                                          │
│                                                                         │
│  4. 整数关口收割期                                                      │
│     特征: 在整数关口反复震荡                                            │
│     动作: 减仓                                                          │
│                                                                         │
│  5. 出货分发期                                                          │
│     特征: 资金费率急速下降，价格回落                                    │
│     动作: 清仓                                                          │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 四、落袋为安规则

```
入场 → 上涨30% → 平仓50% → 锁定利润
                ↓
         剩余50%仓位 → 跟踪止损(回撤5%平仓)
```

| 阶段 | 动作 | 仓位 |
|------|------|------|
| 入场 | 初始仓位 | 100% |
| 达到30%止盈 | 平仓50% | 剩余50% |
| 剩余仓位回撤5% | 全部平仓 | 0% |

---

## 五、资金费率分级管理

| 费率区间 | 动作 | 说明 |
|----------|------|------|
| < 0% | 安全 | 空头主导，可能反弹 |
| 0% - 0.15% | 正常 | 可持有 |
| 0.15% - 0.5% | 预警 | 关注，准备减仓 |
| 0.5% - 1.0% | 减仓 | 减30%仓位 |
| > 1.0% | 清仓 | 强制全部平仓 |
